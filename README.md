# L2D-Chess-A-variant-engine
A engine made in Love2D made with the idea for making variants. I made this in like three hours with only my phone so enjoy.

Features:

- any piece can be royal
- rotating pieces for local play
- screen fits for all devices
- chess

Things to be added:

- special moves (castling, en passant, promotion)
- dragging
- checkmate screen (for now it's just a sound)
